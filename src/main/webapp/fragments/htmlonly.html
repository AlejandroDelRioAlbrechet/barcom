This is an example fragment using only static html content. There is another example fragment available that is more dynamic. You'll notice that you are able to use the browser back button to return to previous fragments.

About fragments

Applications built using MAT are divided intro fragments that can be loaded into the web page at your convenience. Fragments can contain static content or they can have a JavaScript counterpart to make them more dynamic. This particular fragment is a static one which is indicated using a special CSS class on the root element. As a convention every fragment has a CSS class on it's root element that is equal to the fragment name.

About snippets

Snippets are repeatable sub-fragments that are cloned and populated as needed. They are present in the base HTML of the fragment and are taken out when the fragment is updated to be shown to the user. The most important goal here is to write the snippets as html instead of having an html string in your JavaScript code.
